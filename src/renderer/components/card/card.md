## card props
+ bg：String

	背景

+ cardImg：String

	卡片图片（LOGO）
	
+ QRCode：String

	二维码
	
+ cardName：String
	
	商家名称
	
+ cardType：String

	卡片类型

+ cardNum：String

	卡号
	
## demo
```html
<card :bg="bg"
	  :cardImg="cardImg"
	  :cardName="cardName"
	  :cardType="cardType"
	  :cardNum="cardNum"
	  :QRCode="QRCode">
</card>
```
